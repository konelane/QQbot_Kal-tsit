
## 掷骰子

*如果真的有神佛上帝，他们会不会掷骰子呢？*

```
.r ndx
```

p.s. 其中，n是骰子数量，x是骰子面数。如.r 1d20则掷出了一颗二十面骰子。



## 查询天文天气

*前往任何地方前，都需要准备精确的气象资料。这是你曾经教给我的。*

```
#天气 龙门 今日
```

p.s. 按照时钟时间排序，查询时间起每3小时一条预报。

## 迷宫

*这种麻烦事以后别来找我，可露希尔或许会感兴趣。*

```

```
