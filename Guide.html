# 掷骰子+决策部门（安科）

如果真的有神佛上帝，他们会不会掷骰子呢？

TIP

.r ndx //使用n个x面骰子

p.s. 其中，n是骰子数量，x是骰子面数。如.r 1d20则掷出了一颗二十面骰子。

#s new 一个议题名称
#s add 议题选项
#s choice

使用罗德岛决策部门对众说纷纭的议题进行决策吧！

# 查询天文天气+地图

前往任何地方前，都需要准备精确的气象资料。这是你曾经教给我的。

#天气 龙门 今日      //查询天气
#地图 罗德岛 999

p.s. 按照时钟时间排序，查询时间起每3小时一条预报。

地图功能最后的数字是地图显示范围，最大1000。

# 四人高级车组建！

博士，车已经备好了！

#22 等/有无                  //建立车队
#22 加入/来了/来/上车/加       //加入本群最新建立的车队
#22 鸽/无了/鸽了              //取消已创建的车队

约各种四人局均可以使用！

注：指令可以用22代替#22，如果自己建立了牌桌，则无法加入其它牌桌，需要先使用22 鸽进行取消。
